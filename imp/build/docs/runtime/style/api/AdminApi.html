3 REST

3.1 /addUser

3.1.1 API Design

URL
/addUser
Method
POST
Request Headers
  • Content-type: application/json
  • Cookie: "value"
Data Params
    {
    "name"  : <value>,
    "email" : <value>,
    "roles" : {"role" : <value> }
    }
Success Response
  • Code: 200

  • ID of User 

        {"UserId" : <value> }
Error Response
  • Code: 401

OR

  • Code: 400 Bad Request

OR

  • Code: 500
Example
    http://localhost:8000/showUsers

3.1.2 Implementation

type AddUserApi = Header "Cookie" String :> "addUser" :> ReqBody '[JSON] User :> Post '[JSON] (Maybe (ResponseUserId))

3.3 /showUserDetails

API to show user details

3.3.1 API Design

URL
/showuserdetails?email=<emailstr>
Method
GET
Headers
  • Cookie : <sessionid>
URL Params
  • "email" : <emailstr>
Success Response
  • Code: 200

  • user with roles 

        {
        "email" : <email>,
        "name"  : <name>,
        "roles" :
            [
                {"role" : <role>}
            ]
    }
Error Response
  • Code: 401 UNAUTHORIZED

OR

  • Code: 400 Bad Request

OR

  • Code: 500
Example
    http://localhost:8000/showUserDetails?email=<email_str>

3.3.2 Implementation

type ShowUserDetailsApi = Header "Cookie" String :> "showUserDetails" :> Capture "email" String :> Post '[JSON] (Maybe (User))
