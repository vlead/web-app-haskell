* REST

** =/logout=

*** API Design

  - URL :: /logout
  - Method :: POST

  - Headers ::
    
    + Content-type: application/json
    + Cookie: <value>

  - Data Params ::  
    #+BEGIN_EXAMPLE
    {
    "userId": <value>,
    "email" : <value>,
    "roles" : {
                "role" : <value>
              }
    }   
    #+END_EXAMPLE
       
  - Success Response ::
    + Code: 200

    + Content :: 
      #+BEGIN_EXAMPLE
      {"userId" : "<value",
       "email"  : "<value",
       "roles"  : {"role" : "<value>"}
      }
      #+END_EXAMPLE     


  - Error Response ::
    + Code: 403 

    OR

    + Code: 400 Bad Request

    OR

    + Code: 500

  - Example ::
    #+BEGIN_EXAMPLE
    http://localhost:8000/logout
    #+END_EXAMPLE

*** Implementation

#+NAME: logout
#+BEGIN_SRC haskell

type LogoutApi = Header "Cookie" String :> "logout" :> ReqBody '[JSON] Session :> Post '[JSON] (Maybe (Session)) 

#+END_SRC

#+NAME: non_admin_routes
